Projects

LetterBooksd

An IOS app for reviewing books similar to how Letterboxd is for reviewing movies. It utilizes a Google Firebase to store Google Books API data of the descriptions of various books that users can search up. They can leave reviews for these books and users can comment on reviews as well. Github Link: https://github.com/joshlee5/LetterBooksd.git

Firebased Blog

A blogging app allowing users to post their thoughts on accounts with email authentication. Both a web application and an IOS mobile app. Users can post and comment on others' posts as well as search for posts. Github Link for web app: https://github.com/joshlee5/Firebased-Blog.git Github Link for IOS app: https://github.com/joshlee5/Firebased-Blog-Mobile.git

Star Wars Guide

An IOS app for fans of Star Wars! It utilizes the Star Wars API to provide information on all of the mainline Star Wars films for fans to use at their leisure! Information includes release dates, cast, and plot synopses. Github Link: https://github.com/joshlee5/Star-Wars-API-App.git
